## Current Project

**[NeuroLens](https://github.com/ali-alsudani/neurolens)** — Classifying psychiatric disorders (schizophrenia, bipolar disorder, major depression) from postmortem brain gene expression data. 607 samples, 9 GEO datasets, SHAP-based feature selection. F1 Macro = 0.716. 7 of 20 model-selected genes confirmed in published psychiatric genetics literature.
